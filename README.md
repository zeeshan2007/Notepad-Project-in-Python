# 📝 Python CLI Notepad App

A simple Command Line Interface (CLI) Notepad application built with Python. This project was created while learning Python fundamentals such as loops, conditions, file handling, and user input.

---

## 🚀 Features

- ✅ Create a new note
- 📖 View saved notes
- ➕ Append new text to existing notes
- 🔍 Search for a specific word in notes
- 🗑️ Clear all notes
- 🚪 Exit the application

---
